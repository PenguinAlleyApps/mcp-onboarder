# Contributing

Thanks for considering a contribution. This repo is part of [PenguinAlley](https://github.com/PenguinAlleyApps) — open-source AI builder tools.

## Issues

Open issues for bugs or feature requests. Include:
- Reproduction steps (exact commands).
- Expected vs actual behavior.
- Environment (OS, language version, related deps).

## Pull requests

Before opening a PR:
1. Fork + branch from `main`.
2. Keep changes scoped — one concept per PR.
3. Run the test suite locally if applicable.
4. Update README if behavior changes.
5. Sign-off commits if you want explicit DCO (not required).

## Code style

- Match existing style — formatter + linter rules in the repo are authoritative.
- No `TODO` comments without an issue link.
- No emojis in code or commits unless explicitly requested.

## Reporting security issues

Email security@penguinalley.com (or open a private GitHub security advisory). Do NOT open public issues for security vulnerabilities.
